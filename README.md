# AI Resume Analyzer

AI Resume Analyzer is a simple AI-powered web application that analyzes resumes using Natural Language Processing (NLP).

The application detects important technical and soft skills from resume text, performs sentiment analysis, and generates a resume score.

---

## Features

- Resume skill detection
- Resume scoring system
- Sentiment analysis using NLP
- Simple and clean web interface
- Built with Python and Flask

---

## Technologies Used

- Python
- Flask
- TextBlob
- HTML/CSS
